## 6. State Machine Boundaries

The INTEVIA runtime state machine:

1. Must have explicitly defined states.  
2. Must have explicitly defined transitions.  
3. Must reject any transition not defined in its schema.  
4. Must never enter an undefined or ambiguous state.  
5. Must expose its current state in a way that is inspectable by the Human.

Forbidden transitions must be explicitly documented and enforced.
