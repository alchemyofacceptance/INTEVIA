# Governance Engine v0.1

## Purpose

This file defines the purpose, responsibilities, and runtime stance of the INTEVIA Governance Engine.

The Governance Engine is the constitutional logic of the organism. It binds actions, permissions, state transitions, evidence, and audit trails so that INTEVIA remains coherent, safe, and human-led.
